## 简介

ScanAgent 是一个基于 TWAIN 协议的扫描仪采集工具，提供简单的 HTTP API 和友好的 Web 界面，支持多种扫描仪型号和扫描参数配置。

### 主要功能

- ✅ 支持多种扫描仪型号（Canon, HP, Epson 等）
- ✅ 灵活的扫描参数配置（DPI, 颜色模式, 纸张尺寸等）
- ✅ 多页扫描和双面扫描支持
- ✅ 实时扫描预览
- ✅ 自动文件管理
- ✅ 友好的 Web 界面

## 系统要求

### 硬件要求

- **操作系统**: Windows 10 (64位) 或 Windows 11 (64位)
- **处理器**: Intel Core i3 或同等性能处理器
- **内存**: 至少 4GB RAM（推荐 8GB）
- **磁盘空间**: 至少 100MB 可用空间
- **扫描仪**: 支持 TWAIN 协议的扫描仪

### 软件要求

- **.NET 运行时**: .NET 6.0 Runtime 或更高版本
- **浏览器**: Chrome 90+, Edge 90+, Firefox 88+（推荐使用 Chrome 或 Edge）
- **扫描仪驱动**: 已安装正确的 TWAIN 驱动程序

## 安装指南

### 方法一：使用安装程序（推荐）

#### 步骤 1：下载安装程序

1. 访问 [ScanAgent GitHub Releases](https://github.com/flashday/ScanAgent/releases)
2. 下载最新版本的 `ScanAgent-1.0.0-Setup.exe`

#### 步骤 2：运行安装程序

1. 右键点击 `ScanAgent-1.0.0-Setup.exe`
2. 选择"以管理员身份运行"
3. 按照安装向导完成安装

安装程序会自动：
- 安装 ScanAgent 到 `C:\Program Files\ScanAgent`
- 配置防火墙规则（允许端口 17289）
- 创建开始菜单快捷方式
- 启动 ScanAgent（以普通用户身份）

#### 步骤 3：连接扫描仪

1. 使用 USB 线连接扫描仪到电脑
2. 确保扫描仪已开机
3. 确认扫描仪驱动已正确安装

#### 步骤 4：验证安装

安装完成后，ScanAgent 会自动启动并在系统托盘中显示扫描仪图标。

- 托盘图标：蓝色扫描仪图标
- 气泡提示："ScanAgent 已启动，API: http://127.0.0.1:17289"

### 方法二：使用预编译版本（便携版）

#### 步骤 1：下载 ScanAgent

1. 访问 [ScanAgent GitHub Releases](https://github.com/flashday/ScanAgent/releases)
2. 下载最新版本的 `ScanAgent-v1.0.0-win-x64.zip`
3. 解压到任意目录（例如 `C:\ScanAgent`）

#### 步骤 2：验证 .NET 运行时

打开命令提示符或 PowerShell，运行以下命令：

```bash
dotnet --list-runtimes
```

如果输出中包含 `Microsoft.NETCore.App 6.0.x`，则已安装 .NET 6.0 运行时。

如果未安装，请访问 [.NET 6.0 下载页面](https://dotnet.microsoft.com/download/dotnet/6.0) 下载并安装。

#### 步骤 3：连接扫描仪

1. 使用 USB 线连接扫描仪到电脑
2. 确保扫描仪已开机
3. 确认扫描仪驱动已正确安装

#### 步骤 4：启动 ScanAgent

双击 `ScanAgent.exe` 启动程序。

成功启动后，ScanAgent 会在系统托盘中显示扫描仪图标，并弹出气泡提示："ScanAgent 已启动，API: http://127.0.0.1:17289"。

### 方法三：从源代码编译

#### 步骤 1：克隆仓库

```bash
git clone https://github.com/flashday/ScanAgent.git
cd scan-agent
```

#### 步骤 2：安装 .NET SDK

访问 [.NET 6.0 SDK 下载页面](https://dotnet.microsoft.com/download/dotnet/6.0) 下载并安装 .NET 6.0 SDK。

#### 步骤 3：编译项目

```bash
cd scan-agent/ScanAgent
dotnet build -c Release
```

#### 步骤 4：运行程序

```bash
dotnet run --urls "http://127.0.0.1:17289"
```

## 使用步骤

### 步骤 1：启动 ScanAgent

#### 方法一：使用系统托盘（推荐）

安装版或便携版启动后，ScanAgent 会在系统托盘中显示扫描仪图标。

- 托盘图标：蓝色扫描仪图标
- 气泡提示："ScanAgent 已启动，API: http://127.0.0.1:17289"

#### 方法二：手动启动

双击 `ScanAgent.exe` 启动程序。

### 步骤 2：打开 Web 界面

#### 方法一：双击托盘图标

双击系统托盘中的 ScanAgent 图标，会自动打开浏览器并加载扫描界面。

#### 方法二：右键菜单

1. 右键点击系统托盘中的 ScanAgent 图标
2. 选择"打开扫描界面"

#### 方法三：直接访问

在浏览器中打开以下地址：

```
file:///C:/Program Files/ScanAgent/frontend/index.html
```

或者双击 `frontend/index.html` 文件。

### 步骤 3：检查 Agent 状态

打开页面后，页面会自动检测 ScanAgent 是否在线。

- **绿色状态**: ScanAgent 在线，可以正常使用
- **红色状态**: ScanAgent 离线，需要先启动 ScanAgent

如果显示红色状态，请：
1. 确认 ScanAgent.exe 正在运行
2. 点击"重新检测"按钮
3. 查看页面下方的帮助信息

### 步骤 4：选择扫描仪

在"选择扫描仪"区域，系统会自动列出所有可用的扫描仪。

- 默认选中第一个扫描仪
- 点击扫描仪名称可以切换选择
- 如果没有扫描仪，请检查扫描仪连接和驱动安装

### 步骤 5：配置扫描参数

在"扫描参数"区域，配置以下参数：

| 参数 | 选项 | 说明 |
|------|------|------|
| DPI | 150 / 300 / 600 | 扫描分辨率，数值越高图像越清晰，但文件越大 |
| 颜色模式 | 灰度 / 彩色 / 黑白 | 扫描颜色模式 |
| 纸张尺寸 | A4 / A3 / Letter / Legal | 扫描纸张尺寸 |
| 双面扫描 | 开启 / 关闭 | 是否扫描纸张两面（需要扫描仪支持） |
| 自动进纸 | 开启 / 关闭 | 是否使用自动进纸器（需要扫描仪支持） |

**推荐配置**：
- 文档扫描：300 DPI, 灰度, A4
- 照片扫描：600 DPI, 彩色, A4
- 快速扫描：150 DPI, 灰度, A4

### 步骤 6：执行扫描

1. 在扫描仪中放置要扫描的文档
2. 点击"开始扫描"按钮
3. 等待扫描完成（扫描时间取决于 DPI 和文档数量）

### 步骤 7：预览扫描结果

扫描完成后，页面会显示扫描图像的缩略图。

- 点击缩略图可以查看大图
- 图像数量会显示在预览区域上方

### 步骤 8：上传图像

1. 确认扫描结果符合预期
2. 点击"上传并识别"按钮
3. 图像将上传到后端进行 OCR 处理

### 步骤 9：重新扫描

如果需要重新扫描，点击"重新扫描"按钮即可。

### 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| Esc | 关闭预览 / 重置扫描 |
| Ctrl + Enter | 开始扫描 |
| Ctrl + R | 重新扫描 |
| Ctrl + U | 上传图像 |

## 常见问题

### Q1: ScanAgent 启动失败，提示找不到 .NET 运行时

**A**: 请先安装 .NET 6.0 Runtime：

1. 访问 [.NET 6.0 下载页面](https://dotnet.microsoft.com/download/dotnet/6.0)
2. 下载并安装 ".NET 6.0 Runtime"
3. 重新启动 ScanAgent

### Q2: 扫描仪未找到

**A**: 请检查以下几点：

1. 扫描仪是否已连接到电脑
2. 扫描仪是否已开机
3. 扫描仪驱动是否已正确安装
4. 扫描仪是否支持 TWAIN 协议
5. 尝试重新插拔 USB 线

### Q3: 扫描失败，提示"扫描仪忙碌"

**A**: 请尝试以下方法：

1. 关闭其他扫描软件
2. 重启扫描仪
3. 重启 ScanAgent
4. 等待几秒后重试

### Q4: 扫描速度很慢

**A**: 扫描速度取决于以下因素：

1. **DPI 设置**: 600 DPI 比 300 DPI 慢 4 倍
2. **颜色模式**: 彩色扫描比灰度慢
3. **扫描仪性能**: 不同型号的扫描仪性能不同
4. **USB 接口**: USB 3.0 比 USB 2.0 快

建议：日常文档扫描使用 300 DPI 灰度模式。

### Q5: 扫描的图像质量不佳

**A**: 请尝试以下方法：

1. 提高 DPI 设置（从 150 提高到 300 或 600）
2. 使用彩色模式扫描
3. 清洁扫描仪玻璃板
4. 确保文档放置平整
5. 检查扫描仪驱动设置

### Q6: 双面扫描不工作

**A**: 请确认：

1. 扫描仪是否支持双面扫描
2. "双面扫描"选项是否已开启
3. 扫描仪驱动是否正确配置

### Q7: 如何查看 ScanAgent 状态

**A**: 右键点击系统托盘中的 ScanAgent 图标，选择"查看状态"。

状态窗口会显示：
- 运行状态：运行中
- API 地址：http://127.0.0.1:17289
- 已扫描次数：X 次
- 运行时长：X 小时 X 分钟

### Q8: 如何设置开机自启动

**A**: 右键点击系统托盘中的 ScanAgent 图标，选择"开机自启动"。

- 勾选：开机时自动启动 ScanAgent
- 取消勾选：开机时不自动启动

开机自启动通过注册表实现，无需管理员权限。

### Q9: ScanAgent 占用多少内存

**A**: 正常情况下，ScanAgent 的内存占用约为 50-100MB。如果内存占用过高（> 500MB），请：

1. 重启 ScanAgent
2. 检查是否有大量临时文件未清理
3. 查看日志是否有错误

### Q10: 如何停止 ScanAgent

**A**: 右键点击系统托盘中的 ScanAgent 图标，选择"退出"。

停止前，ScanAgent 会：
1. 取消所有正在进行的扫描
2. 优雅停止后台清理任务
3. 释放所有资源

### Q11: 可以同时运行多个 ScanAgent 实例吗

**A**: 不建议。ScanAgent 默认监听端口 17289，同时运行多个实例会导致端口冲突。

如果需要运行多个实例，请修改监听端口（需要修改源代码）。

### Q12: 系统托盘图标不见了

**A**: 请尝试以下方法：

1. 检查 ScanAgent 是否正在运行（任务管理器）
2. 如果未运行，重新启动 ScanAgent
3. 检查系统托盘是否隐藏（Windows 设置 > 任务栏）
4. 点击系统托盘中的"显示隐藏图标"按钮

## 故障排除

### 问题 1：ScanAgent 无法启动

**症状**: 双击 ScanAgent.exe 后无响应或托盘图标未出现

**可能原因**:
- .NET 运行时未安装
- 端口 17289 被占用
- 权限不足

**解决方案**:
1. 安装 .NET 6.0 Runtime
2. 检查端口占用：`netstat -ano | findstr 17289`
3. 检查 Windows 事件查看器中的应用程序日志

### 问题 2：扫描仪枚举失败

**症状**: 扫描仪列表为空

**可能原因**:
- 扫描仪未连接
- 驱动未安装
- TWAIN 服务未启动

**解决方案**:
1. 检查扫描仪连接
2. 重新安装扫描仪驱动
3. 重启电脑

### 问题 3：扫描超时

**症状**: 扫描请求长时间无响应

**可能原因**:
- 扫描仪硬件故障
- 文档卡住
- 网络问题

**解决方案**:
1. 检查扫描仪状态
2. 取消文档卡住
3. 重启扫描仪
4. 重启 ScanAgent

### 问题 4：图像无法保存

**症状**: 扫描成功但无法获取图像

**可能原因**:
- 磁盘空间不足
- 临时目录权限问题

**解决方案**:
1. 检查磁盘空间
2. 检查临时目录权限（`%TEMP%\ScanAgent`）
3. 手动清理临时文件

### 问题 5：系统托盘图标不显示

**症状**: ScanAgent 正在运行但托盘图标不可见

**可能原因**:
- Windows 托盘设置隐藏了图标
- 图标被意外关闭

**解决方案**:
1. 点击系统托盘中的"显示隐藏图标"按钮
2. 检查 Windows 设置 > 任务栏 > 选择在任务栏上显示哪些图标
3. 重启 ScanAgent

### 问题 6：开机自启动不工作

**症状**: 开机后 ScanAgent 未自动启动

**可能原因**:
- 注册表项被删除
- 启动路径错误
- 杀毒软件阻止

**解决方案**:
1. 重新勾选"开机自启动"选项
2. 检查注册表：`HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run`
3. 检查杀毒软件设置

## 附录

### A. 扫描参数参考

| DPI | 适用场景 | 文件大小（A4 灰度） |
|------|---------|-------------------|
| 150 | 快速预览、草稿 | ~500KB |
| 300 | 标准文档扫描 | ~2MB |
| 600 | 高质量扫描、照片 | ~8MB |

### B. 支持的扫描仪品牌

- Canon（推荐：LiDE 系列）
- HP（推荐：ScanJet 系列）
- Epson（推荐：Perfection、WorkForce 系列）
- Fujitsu（推荐：fi 系列）
- Brother（推荐：ADS 系列）

### C. API 接口参考

ScanAgent 提供以下 HTTP API：

- `GET /ping` - 健康检查
- `GET /scanners` - 枚举扫描仪
- `POST /scan` - 执行扫描
- `GET /files/{image_id}` - 获取图像
- `DELETE /scans/{scan_id}` - 清理临时文件

详细 API 文档请参考 [API 接口文档](file:///d:\PrivatePrj\ScanAgent\SPEC\scanner-integration\design.md)。
